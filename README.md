# 執行與部署您的 AI Studio 應用程式

這裡包含您在本地端執行應用程式所需的一切。

在 AI Studio 中查看您的應用程式：https://ai.studio/apps/drive/1uslD9jnLjdtnMcV5keZZ_HCC0QKjp9y-

## 在地端執行

**先決條件：** Node.js

1. 安裝相依套件：
   `npm install`
2. 在 [.env.local](.env.local) 中將 `GEMINI_API_KEY` 設定為您的 Gemini API 金鑰
3. 執行應用程式：
   `npm run dev`
